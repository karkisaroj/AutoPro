# AutoPro Garage System 🏎️✨

Welcome to the frontend repository for the **AutoPro Garage System**, a premium, modern web application designed to bring transparency, efficiency, and high-quality service to the auto repair industry in Nepal.

## 🌟 About the Project

AutoPro Garage System is a comprehensive shop management and customer booking portal. We provide a seamless digital experience where car owners can easily view services, find certified technicians, and book appointments, while the garage staff can efficiently handle the administrative workload.

## 🚀 Features

- **Premium UI/UX**: An immersive landing page featuring modern aesthetics, smooth micro-animations, and a responsive glassmorphism design.
- **Service Catalog**: Detailed views of our specialized offerings including Engine Diagnostics, Brake Replacements, and EV Battery Service.
- **Modern Architecture**: Fast, lightweight component-based architecture optimized for performance and scalability.
- **Interactive Forms**: A streamlined contact interface integrated with interactive design language.

## 🛠️ Technology Stack

- **Frontend Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Routing**: [React Router DOM v7](https://reactrouter.com/)
- **Styling**: Vanilla CSS (Custom Design System, utilizing CSS Variables, Flexbox, and Grid)
- **Icons**: [Lucide React](https://lucide.dev/)

*Note: This project purposefully avoids utility-class frameworks like Tailwind CSS to maintain a bespoke, highly customized CSS design system optimized for absolute control over the UI components.*

## ⚙️ How to Run Locally

1. **Install Dependencies**
   Make sure you have Node installed, then run:
   ```bash
   npm install
   ```

2. **Start the Development Server**
   ```bash
   npm run dev
   ```

3. **View in Browser**
   Open `http://localhost:5173` (or the port provided in your terminal) to view the application.
